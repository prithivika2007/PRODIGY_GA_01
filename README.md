# PRODIGY_GA_01 - Text Generation with GPT-2

## Objective
Build a text generation model using GPT-2 that generates coherent and contextually relevant text based on a given prompt.

---

## Project Overview
This project demonstrates text generation using the pre-trained GPT-2 language model from Hugging Face Transformers. A custom dataset containing motivational quotes was used to fine-tune the model, enabling it to generate meaningful and context-aware text.

---

## Features
- Load the GPT-2 Small model
- Fine-tune on a custom motivational quotes dataset
- Generate text from user-defined prompts
- Produce coherent and contextually relevant outputs

---

## Technologies Used
- Python
- Google Colab
- Hugging Face Transformers
- PyTorch

---

## Dataset
A custom text dataset (`data.txt`) containing motivational quotes was used for fine-tuning the GPT-2 model.

---

## Sample Output

**Prompt**
```
Believe in yourself because
```

**Generated Text**
```
Believe in yourself because everyone else has already done so. You don't have to believe that every day they'll succeed, and one day they'll be happy they do.
```
