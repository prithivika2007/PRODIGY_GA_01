# PRODIGY_GA_01 - Text Generation with GPT-2

## Task
Fine-tune GPT-2 to generate coherent and contextually relevant text based on a given prompt.

## What I did
- Loaded GPT-2 (small) from HuggingFace
- Fine-tuned it on a custom motivational quotes dataset
- Generated text using different prompts

## Sample Output
Prompt: "Believe in yourself because"
Output: "Believe in yourself because you're a true leader. Take it easy and be honest..."

## Tools Used
- Python
- HuggingFace Transformers
- Google Colab
